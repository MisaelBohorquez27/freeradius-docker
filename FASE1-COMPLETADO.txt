PASO 1: Generar Certificados
-----------------------------
bash generate-certs.sh

Esto crea:
  - certs/ca.crt (Certificado de CA)
  - certs/ca.key (Clave privada - SECRETO)
  - certs/server.crt (Certificado del servidor)
  - certs/server.key (Clave privada del servidor)


PASO 2: Configurar Omada Controller
-----------------------------------
En clients.conf, cambiar IP actual (192.168.1.100) a la real:

   client omada_controller_primary {
       ipaddr = [IP_REAL_DE_TU_OMADA_CONTROLLER]
       secret = RadSecure@2026.prod.enterprise.freeradius
   }

IMPORTANTE: El secret debe ser el MISMO en Omada y FreeRADIUS


PASO 3: Validar Configuración
------------------------------
bash validate-phase1.sh

Esto verificará:
  ✓ Estructura de archivos
  ✓ Configuración de seguridad
  ✓ Volúmenes en docker-compose
  ✓ Certificados generados


PASO 4: Levantar el Servicio
-----------------------------
mkdir -p logs
docker-compose build
docker-compose up -d

Verificar:
docker ps | grep freeradius
docker logs -f freeradius
